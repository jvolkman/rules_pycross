## `default_enable_implicit_namespace_packages`

Flag: `--@rules_pycross//pycross/config_settings:default_enable_implicit_namespace_packages`
Type: `bool`
Default: `false`

When enabled, [`pycross_wheel_library.enable_implicit_namespace_packages`](./rules.md#pycross_wheel_library-enable_implicit_namespace_pkgs) will default to `True`.
